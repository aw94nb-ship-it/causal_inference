# Resources

## Textbooks

### Foundational
- **Angrist, J. D., & Pischke, J. S. (2009).** *Mostly Harmless Econometrics*
  - Practical guide to applied causal inference

- **Morgan, S. L., & Winship, C. (2014).** *Counterfactuals and Causal Inference*
  - Comprehensive overview of the field

- **Imbens, G. W., & Rubin, D. B. (2015).** *Causal Inference for Statistics, Social, and Biomedical Sciences*
  - Detailed treatment of potential outcomes framework

### Advanced
- **Pearl, J. (2009).** *Causality: Models, Reasoning, and Inference*
  - Comprehensive treatment using DAGs and structural causal models

- **Hern√°n, M. A., & Robins, J. M. (2020).** *Causal Inference: What If*
  - Modern epidemiological perspective (free online!)

- **Cunningham, S. (2021).** *Causal Inference: The Mixtape*
  - Applied focus with code examples (free online!)

## Software and Tools

### Python
- **DoWhy**: Microsoft's causal inference library
- **CausalML**: Uber's machine learning library for causal inference
- **EconML**: Microsoft's library for heterogeneous treatment effects
- **PyMC**: Bayesian causal inference

### R
- **MatchIt**: Matching methods
- **mediation**: Mediation analysis
- **rdrobust**: Regression discontinuity
- **synth**: Synthetic control methods

## Papers and Articles
